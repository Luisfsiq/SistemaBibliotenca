Sistema de Gestão de Biblioteca (Java Console) 📚
Este projeto é uma aplicação de console desenvolvida para gerir o fluxo de livros, autores e empréstimos de uma biblioteca. O foco principal foi a aplicação prática dos conceitos de Programação Orientada a Objetos (POO) e a organização de lógica de negócios em Java.

O desenvolvimento deste projeto consolidou conhecimentos adquiridos na disciplina de Programação de Computadores no UNIPÊ e no micro-certificado de Fundamentos de Java da Faculdade de Tecnologia Rocketseat.

🚀 Funcionalidades
Gestão de Acervo: Visualização de livros e a sua disponibilidade em tempo real.

Fluxo de Empréstimo: Registo de empréstimos associando clientes a livros específicos.

Sistema de Devolução: Atualização dinâmica do estado do livro e registo da data de entrega.

Histórico de Transações: Relatório detalhado de todos os empréstimos realizados, pendentes ou concluídos.

Base de Autores: Listagem de informações sobre os autores registados no sistema.

🛠️ Tecnologias Utilizadas
Java 17+: Linguagem principal.

JDK/JRE/JVM: Compreensão do ambiente de execução.

Java Time API (java.time): Gestão rigorosa de datas de empréstimo e devolução.

Estruturas de Dados (ArrayList): Manipulação de coleções em memória.

🧩 Conceitos de Engenharia de Software Aplicados
POO (Programação Orientada a Objetos): Implementação de Classes, Atributos, Métodos e Encapsulamento (Getters/Setters).

Modularização: Separação de responsabilidades entre as classes Livro, Autor, Emprestimo e a classe principal Biblioteca.

Lógica de Fluxo: Utilização de estruturas de repetição e controlo (switch-case, while, if/else) para criar uma interface interativa via console.

🔧 Como Executar
Certifique-se de que possui o JDK 17 ou superior instalado.

Clone o repositório:

Bash
git clone https://github.com/seu-usuario/sistema-biblioteca-java.git
Navegue até a pasta do projeto e compile os ficheiros:

Bash
javac *.java
Execute a aplicação:

Bash
java Biblioteca
